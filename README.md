Discord bot for discord servers:)

## Requirements
 - `NodeJS 6`
 - `Discord.js 11.5`

## Installation


## Usage
 - Create 'bot.json' file with this content: 
 `{
    "youtube_token":"",
    "discord_token":""
} 
`
 - Start index.js with command `node index.js`
